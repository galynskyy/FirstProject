Центрирование по горизонтали

Здесь не рассмотрены варианты выравнивания по вертикали с помощью: padding and margin!

inline или inline-block элементы внутри родительского можно центрировать так:

      .wrap {
        text-align: center;
      }

Блочные элементы можно центрировать так, задав им свойство:

    .item {
      margin: 0 auto;
    }

Любые (блочные, строчные и т.д.) элементы можно центрировать с помощью flexbox, указав родителю:

    .wrap {
      display: flex;
      justify-content: center;
    }

Центрирование внутри таблиц для inline элементов:

    .wrap {
      display: table;
    }

    .inner {
      display: table-cell;
      text-align: center;
    }

Выравние с помощью позиционирования (absolute, fixed), например: absolute:

    .wrap {
      position: relative;
    }

    .item {
      position: absolute;
      top: 0;
      right: 0;
      bottom: 0;
      left: 0;
      width: 140px;
      height: 80px;
      margin: 0 auto;
    }

Центрирование по вертикали

Любые (блочные, строчные и т.д.) элементы можно центрировать с помощью flexbox, указав родителю:

    .wrap {
      display: flex;
      justify-content: center;
      align-items: center;
    }

С помощью line-height если однострочный текст (можно еще добавить white-space)

    .wrap {
      text-align: center;
    }

    .item {
      display: inline-block;
      line-height: 200px;
    }

С помощью line-height для многострочного текста (если высота родителя известна, а центрируемого элемента – нет)

    .wrap {
      height: 200px;
      line-height: 200px;
      text-align: center;
    }

    .item {
      display: inline-block;
      vertical-align: middle;
      line-height: normal;
    }

inline, inline-block элементы с помощью элемента призрака (:before, :after) (например: текста > 1 строки)

    .wrap {
      text-align: center;

      &:before {
        content: '';
        display: inline-block;
        height: 100%;
        vertical-align: middle;
      }
    }

    .item {
      display: inline-block;
      vertical-align: middle;
    }
Lorem ipsum dolor sit amet, consectetur adipisicing elit.

Выравнивание по вертикали с помощью имитации таблицы:

    .wrap {
      display: table; /* table нужен, чтобы не задавить явный размер у table-cell. */
    }

    .table-cell {
      display: table-cell;
      text-align: center;
      vertical-align: middle;
    }

Выравние, если неизвестны размеры блока, с помощью позиционирования (absolute, fixed), например: absolute

    .wrap {
      position: relative;
    }

    .item {
      position: absolute;
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
    }

      top: 50%;
